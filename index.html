O que é o Weka?

O Weka é um produto da Universidade de Waikato(Nova Zelândia) e foi implementado pela primeira vez em sua forma moderna em 1997. Ele usa a GNU General Public License (GPL). O software foi escrito na linguagem Java™ e contém uma GUI para interagir com arquivos de dados e produzir resultados visuais como tabelas. Ele também tem uma API geral, assim é possível incorporar o WEKA, como qualquer outra biblioteca, a seus próprios aplicativos para fazer coisas como tarefas de mineração de dados automatizadas no lado do servidor. O software tem como objetivo agregar algoritmos de diferentes paradigmas dedicada à chamada Machine Learning,e, testa indutivamente os padrões dos dados fornecidos,gerando hipóteses e teorias sobre as informações em questão.
